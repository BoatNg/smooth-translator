TransIt
===========

一款简单方便的划词翻译扩展。

功能列表：

- 页面英文划词翻译 
- 连续对多个单词进行划词翻译
- 按住 Shift 键可以对超链接中的文本进行划词翻译
- 缓存已经查询过的单词的释义，下次查询更快速
- 调整页面划词翻译结果显示的时间长短

技术支持：

- 产品开发团队 http://veaku.com/
- 翻译技术来源 http://fanyi.youdao.com/openapi?path=data-mode
