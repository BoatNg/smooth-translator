TransIt
===========

让划词翻译更简单

#### 功能列表

- 页面英文划词翻译 
- 连续对多个单词进行划词翻译
- 按住 Shift 键可以对超链接中的文本进行划词翻译
- 调整页面划词翻译结果显示的时间长短
- 适应更多的页面，包括 Iframe 嵌套
- 支持在窗口边缘和选词附近两种方式显示翻译结果
